Loyalty Wallet Widget Integration

This page demonstrates how to integrate the Loyalty Wallet widget into your website.

Integration Steps:

  1. Add the script tag to your HTML:
<script src="https://your-domain.com/loyalty-wallet-widget-with-auth.js?orgId=YOUR_ORG_ID" type="text/javascript" crossOrigin="anonymous" async="true" defer="true"></script>
Note: Replace YOUR_ORG_ID with your actual organization ID and update the script URL to point to your hosted widget file.

Authentication:

The widget automatically handles authentication using built-in credentials. No manual token management required!

Note: The widget automatically authenticates with the backend using predefined credentials. Organizations can customize the authentication credentials in the widget code if needed.

Widget Features:

  • ✅ Apple Wallet integration
  • ✅ Google Wallet integration
  • ✅ QR code generation
  • ✅ Responsive design
  • ✅ Authentication handling
  • ✅ Error handling

Automatic Authentication:

  • ✅ No manual token management required
  • ✅ Automatic login with backend
  • ✅ JWT token handling
  • ✅ Token persistence for future use

Live Demo:

The loyalty wallet widget should appear in the bottom right corner of this page.

The widget will automatically authenticate and be ready to generate QR codes!

API Endpoints Used:

  • GET /wallet/apple/qrCode - Generate Apple Wallet QR code
  • GET /wallet/google/qrCode - Generate Google Wallet QR code

Both endpoints require a valid Bearer token in the Authorization header.
